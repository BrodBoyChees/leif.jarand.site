Lage group polecy som lager snarveg til Truenas mapper

    
    Opne server manager og trykk tools oppe i høgre hjørne.
    Frå tools menyen velg group polecy managment
    Opne Forest > Domains > ditt domene.
    Høgreklikk på (domene navn).
    
    Trykk på "create a GPO" og gi den navnet "elevar-snarveg".
    Høgreklikk på polecyen du nettop lagde og trykk "Edit".
    
    
    Inne på "Group polecy managment editor" går du til User Configuration > Prefrences > Windows Settings > Shortcuts.
    Høgreklitt på "Shortcuts" trykk "new" og "shortcut"
    
    Velg "create" på "Action" feltet.
    Gi snarvegen navnet "Felles" og sett "location" til "desctop".
    Set "target path" til "\\truenas.lab.local\elevar".
    Deretter trypp på "Apply" og "ok"
    
    Lag ein til snarveg med samme konfigurasjon men der skal navnet vere privat mappe og "target path" vere "\\truenas.lab.local\%username%".
    
    Lag ein til polecy som heiter lererar-snarveg og bytt up "target path" i den fyste sanrvegen med "\\truenas.lab.local\lererar".
    
    Du må ha lagt alle elevar og lererar under kvar sine OUar i AD. Høgreklikk på OUen elev eller lerar brukarane er i og velg "link an existing GPO" og velg elevar-snarveg eller lererar-snarveg. Gjer dete på både elev og lerar OUen.
    
    Neste gong du loggar inn med ein klient skal det komme to snarvegar på skrivebordet som går til ei privat mappe og ei felles mappe. 
    Om du ikkje vill logge ut og inn igjen kan du gjere "gpupdate" i command Prompt appen.
    
    Om du har gjort det samma som meg skal alt funke men om du har gitt sharesene andre navn å du endre det i "target path" til snarvegen.
