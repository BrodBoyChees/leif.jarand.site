Active direcotry

Før du begynner må du forsikre om at Truenas og Windows server er kopla på internett. 
Truenas og Windows server treng å ha same klokkeslett for at Truenas skal kopla seg på AD. 
Truenas treng og internett for at AD brukarar skal få komme seg in på netverksresursane. 
Begynn med å sette opp AD og lage grupper og brukarar som du treng.


Tidssone

Inne på Truenas nettsida går du på System > General i menyen til venstre.

Under "Timezone" skriv du "Europe/Oslo"  Windows server må og vere på denne tidssonene. 
Trykk deretter på "save".
For å sjekke at klokka er rett kan du gå i Shell i sidemenyen og skrive "date" i terminalen.


DNS og Gateway

Truenas må ha Windows server som DNS server for å få kontakt med domenet. 
Husk å installere DNS på «Windows server. 
Gå til Network > Global Configuration i sidemenyen.

Her skriv du inn IP adressa til windows serveren din i feltet "Nameserver 1".
Her må du og skrive inn default gateway. Du skal skrive inn IP adressa til ruteren din under "IPv4 Default Gateway".
For å sjekke at DNS virkar kan du gå til Shell i sidemenyen og skrive "ping (ditt-domene)".


Active Directory

Gå til Direcotory Services > Active Directory i sidemenyen.

Skriv inn domenenavnet ditt under "Domanin Name".
Skriv Administrator under "Domain Account Name".
Skriv passorded til Administrator kontoen under "Doman Account Password" 
(Det er det samme passorded som du bruker til å logge på windows server)
Du skal og hake av avmerkingsboksen under.
Trykk "save" og deretter "Rebuild Directory Service Cache".

Oppe i høgre hjørne skal det no ha komt eit nytt ikon. Trykk på det få å sjå statusen til AD.

For å sjekk at Truenas kan sjå brukarar og grupper går du till Shell i sidemenyen og skriv "wbinfo -u" for å liste alle brukarar eller "wbinfo -g" for grupper.

Om du legg til eller fjernar brukarar/grupper må du trykker på "Rebuild Directory Service Cache" i Servises > Active Directory. Dette oppdaterer cache til Truenas.
